\section{Hardware}
\subsection{Raspberry Pi}
Der Raspberry Pi ist ein Einplatinencomputer in der Größe einer Kreditkarte. Entwickelt wurde er von der britischen Stiftung Raspberry Pi Fundation um inbesondere jungen Leuten eine kostengünstige Platform zum aneignen von Programmierkenntnissen zu bieten. Für meine Bachelorthesis habe ich den Raspberry Pi 2 Model B  genutzt. Der Verkaufspreis für dieses Modell beträgt in Deutschland etwa 38~EUR (11/2015), ohne zum Betrieb nötiges Zubehör wie Netzteil oder microSD-Karte.
